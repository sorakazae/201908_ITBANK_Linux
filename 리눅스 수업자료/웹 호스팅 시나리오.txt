웹 호스팅 시나리오..
1. 웹 호스팅 업체 : 10.10.0.100
2. 도메인 : bong.com
shop.bong.com -> 10.10.0.50 웹서비스
www.bong.com -> 10.10.0.200 웹서비스
2.웹 서버 : 10.10.0.200, 10.10.0.50 서비스
자체 dns 서버는 사용 하지 않음
3. 도메인 세팅 : 웹 호스팅 업체의 DNS에 서관리 
호스팅 업체 홈페이지에 접속해서 호스트 추가와  애플리케이션 서비스가 이루어지는 호스트 컴퓨터에 ip 번호를 세팅해서 서비스가 되도록 설정..
www.bong.com 10.10.0.200 
shop.bong.com 10.10.0.50 
ping www.bong.com -> 실제 호스트 컴퓨터 운용
ping shop.bong.com -> 실제 컴퓨터 운영중이지 않으면 ping 응답 못함
