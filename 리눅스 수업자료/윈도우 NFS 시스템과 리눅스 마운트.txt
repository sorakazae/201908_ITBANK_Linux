windows 2008R2 에서 NFS 공유파일 시스템 생성 하고 리눅스에서 마운트

1.관리도구 ->서버관리자 -> 역활 ->역활 추가 ->파일 서비스 체크-> 네트워크 파일 시스템용 서비스 체크 -> 설치

2. 공유폴더 속성-> NFS 공유 ->NFS 공유 관리 클릭 ->서버 인증 없음 만체크 : 익명 엑세스 허용 체크/매핑되지 않은 사용자 엑세스 사용 체크 -> 권한 클릭 : 루트 엑세스 허용 체크 -> 확인

3. 리눅스
mount -t nfs 10.10.0.20:/smb /smbmount
